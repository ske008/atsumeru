# アツメル - プロジェクト進捗報告書

最終更新: 2026年2月15日

## プロジェクト概要

**アツメル**は、イベントの出欠管理と集金を一括で行えるWebアプリケーションです。

### 主な機能

- **イベント作成**: タイトル、日時、場所、メモを設定してイベントを作成
- **出欠管理**: URLを共有して参加者が出欠を回答
- **集金機能**: 参加者ごとの支払い状況を管理
- **PayPay連携**: 送金先URLを設定して、参加者が直接送金可能

### 技術スタック

- **フロントエンド**: React (CDN版)
- **データ保存**: LocalStorage
- **スタイル**: カスタムCSS（デザイントークンベース）
- **構成**: Next.js App Router（`app/`）構成
